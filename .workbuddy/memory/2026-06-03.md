## AI群聊MVP - ai-group-chat/

构建了完整的 AI 群聊 MVP 项目（Node.js 后端 + Vite+React 前端），路径：`ai-group-chat/`

### 技术栈
- 后端：Node.js + Express + TypeScript + WebSocket (ws) + OpenAI SDK
- 前端：Vite + React + TypeScript + Tailwind CSS

### 核心模块
- `server/src/director.ts` — Director Agent（控制发言节奏、idle触发）
- `server/src/events.ts` — Event Engine（8种群事件）
- `server/src/characterAgent.ts` — Character Agent（调用OpenAI生成角色发言）
- `server/src/memory.ts` — Memory Manager（最近20条上下文）
- 前端 WebSocket hook + 微信风格聊天气泡 + 打字动效

### 运行方式
```bash
cd ai-group-chat && cp server/.env.example server/.env  # 填Key
npm install && npm run install:all && npm run dev
```

## 旅伴圈产品原型 travel-tribe.html

构建了完整的旅行社交产品HTML原型，文件路径：`travel-tribe.html`

### 核心功能实现
- **微信式群聊UI**：状态栏、聊天列表、群聊界面、底部导航，完整还原WeChat感
- **AI角色成员**：旅神·小京（AI向导）+ 摄影·阿光（AI摄影伙伴），有AI色环标识，点击可查看AI画像
- **群事件系统**：投票决策卡片，AI主动发起，成员实时参与
- **旅行日记**：可发布日记卡片，AI角色自动评论互动
- **时间胶囊**：封存/解锁机制，进度条展示
- **成员画像面板**：AI性格特征（探索欲/话唠度/主动性）+ 贡献记录
- **定时AI活动**：8s/15s/22s 自动触发AI Toast通知和新消息，模拟"活着的群"
- **快捷操作栏**：打卡位置/写日记/存胶囊/群事件

### 设计取向
- 不复制微信，但用微信式UI语言
- 突出AI角色的主动性（主动推送路线、拍摄时机、历史故事）
- 时间胶囊作为情感锚点
- AI角色有明确个性和贡献可视化
